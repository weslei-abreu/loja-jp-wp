A starter theme called Thembay Elementor, or underscores.

== Description ==

Thembay Elementor is a WooCommerce WordPress Theme that is flexible and customizable for setting and changing any elements within a minute via Powerful Theme Options, you also can customize Google fonts without code very easily and simply.


== Installation ==

1. Upload the entire tbay-elementor folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
